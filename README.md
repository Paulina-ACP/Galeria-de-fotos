Galeria de Fotos

Este projeto foi desenvolvido como parte de um teste técnico de um processo seletivo do qual estou participando.
O objetivo foi criar uma galeria de fotos responsiva, com foco em Mobile First, usabilidade e organização do código.


Funcionalidades

Layout responsivo (Mobile First)

Exibição dinâmica das imagens

Busca de fotos pelo nome

Visualização da imagem em tela cheia ao clicar

Botão para fechar e retornar à galeria


Tecnologias Utilizadas

HTML5

CSS3

JavaScript (puro)
